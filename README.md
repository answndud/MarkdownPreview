# 📝 Markdown Preview - 마크다운 미리보기

실시간 마크다운 에디터 및 미리보기 도구입니다.

## ✨ 기능

- ⚡ **실시간 변환**: 입력과 동시에 HTML 미리보기
- 🐙 **GitHub Flavored Markdown (GFM)**: GitHub 스타일 마크다운 완벽 지원
- 🎨 **코드 문법 하이라이팅**: 다양한 프로그래밍 언어 지원
- 📋 **템플릿 제공**: README, 블로그, 할일 목록 등
- 📄 **복사 기능**: Markdown / HTML 복사
- ⬇️ **다운로드**: .md 파일로 저장
- 💾 **자동 저장**: LocalStorage에 내용 보존
- 🌓 **라이트/다크 모드**: 원클릭으로 테마 변경

## 📝 지원 문법

### 기본 문법

| 문법 | 마크다운 | 결과 |
|------|----------|------|
| 제목 | `# H1` ~ `###### H6` | 6단계 제목 |
| 굵게 | `**텍스트**` | **텍스트** |
| 기울임 | `*텍스트*` | *텍스트* |
| 취소선 | `~~텍스트~~` | ~~텍스트~~ |
| 링크 | `[텍스트](URL)` | [텍스트](URL) |
| 이미지 | `![alt](URL)` | 이미지 |
| 인용 | `> 인용문` | 인용문 |
| 수평선 | `---` | 구분선 |

### GitHub 확장 문법

```markdown
# 체크박스
- [x] 완료된 항목
- [ ] 미완료 항목

# 테이블
| 헤더 1 | 헤더 2 |
|--------|--------|
| 셀 1   | 셀 2   |

# 코드 블록
```javascript
console.log('Hello!');
```
```

| 키 | 기능 |
|----|------|
| `Tab` | 4칸 들여쓰기 |
| `Esc` | 전체화면 종료 |

## 🚀 사용 방법

1. 왼쪽 에디터에 마크다운 입력
2. 오른쪽에서 실시간 미리보기 확인
3. 필요시 템플릿 선택하여 시작
4. 완성 후 MD/HTML 복사 또는 다운로드
